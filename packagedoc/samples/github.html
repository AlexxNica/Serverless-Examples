/whisk.system/github

Package which contains actions and feeds to interact with Github (version: 0.0.191)

Parameters

Name Default Type Required
endpoint openwhisk.ng.bluemix.net

Actions

This package has no actions.

Feeds

Name Parameters (* is required)
webhook (Creates a webhook on GitHub to be notified on selected changes)

Sample Input: 

{
	"accessToken": "123ABCXYZ",
	"events": "push,delete,pull-request",
	"repository": "myRepository or myOrganization/myRepository",
	"username": "myUserName"
}
  • username* (Your GitHub username)
  • repository* (The name of a GitHub repository)
  • accessToken* (A webhook or personal token)
  • events* (A comma-separated list)
